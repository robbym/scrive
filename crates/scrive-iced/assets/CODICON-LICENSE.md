# Codicon

`codicon.ttf` in this directory is the [Codicon](https://github.com/microsoft/vscode-codicons)
icon font — the icon set used by Visual Studio Code.

- **Version:** 0.0.45
- **Source:** https://github.com/microsoft/vscode-codicons (npm `@vscode/codicons`)
- **Icons © Microsoft Corporation.**
- **Icons licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).**
- Font tooling / code is MIT.

scrive bundles the font (rather than depending on the system) so the editor's
fold-gutter chevrons and find-bar controls render VS Code's exact glyphs on any
platform. Glyphs currently used: `chevron-right` (U+EAB6), `chevron-down`
(U+EAB4), `arrow-up` (U+EAA1), `arrow-down` (U+EA9A), `close` (U+EA76).
